# Prakriti-Chatbot-
This is an ayurvedic chatbot to determine your prakriti using some predefined questiones.
The chatbot will provide some questionnaire one by one, and user has to answer those questions .
➢After the response is collected from the user and is splitted into individual words and keywords content (available in the dataset) in the user input is being checked
➢After checking , the data is being stored in management code block which is specifically built to calculate percentages.
➢Then the bot checks if the question is the last among the list of questionnaires , if not then proceed to the next question , and bot enquires user with the next question
➢If yes , the bot will show the percentages and the pie chart diagram .
